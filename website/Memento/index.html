La ligne de commande

IL en va sans dire que pour naviguer dans votre ordinateur, un Terminal est nécessaire. Vous le trouverez nativement sous la distribution de Linux Ubuntu et les autres, ou sur Mac.
Il vous faudra sous windows installer une distribution Linux pour utiliser le Terminal qui se nommera "Bash".
Pour savoir comment installer bash sur windows, cliquez ICI

Les Bases

Pour ouvrir le manuel de la liste de commande on utilise la commande : man ls
Ensuite, si on souhaite connaître la liste des dossiers et des fichiers contenu dans le dossier où on se trouve, on utilise la commande : ls

Si l'on souhaite le contenu du dossier courant avec les fichiers cachés, la commande est la suivante :ls -a


On peut également afficher, toujours dans le dossier courant, le contenu en version longue sans les fichiers cachés. La commande est la suivante: ls -l
Enfin, toujours dans le dossier courant, on peut combiner les deux commandes précédentes pour afficher le contenu en version longue AVEC les fichiers cachés. Attention, combo : ls -al

Pour afficher le chemin du dossier courant, qui répond à cette question: "Mais bon sang, où je suis dans ce système?!", la commande très utile est pwd


Pour se déplacer, très utiles me direz vous, rien de plus simple que cette commande ci: cd "nom-du-fichier"
Et por revenir en arrière? Toujours aussi simple. Voyez vous même: cd ..
L'ajout des deux poits ".." permet de revenir vers le dossier dit "parent".

Super Utilisateur et autres joyeusetés

Se mettre en super utilisateur "su, il y a deux possibilités qui sous leurs apparente ressemblance sont bien différentes


La commande sudo -s permet de passer en Super Utilisateur (su) MAIS de rester dans le dossier où on se trouve.
Son copain, la commande sudo -i renvoit lui vers le dossier Root de la machine.
Cette différence est très IMPORTANTE et n'oubliez jamais que lorsque vous êtes en Super Utilisateur, tout ce que vous faites est très, très important, vous pouvez détruires des informations sur votre système aussi facilement qu'un click.
A manipuler avec grande précaution. On vous aura prévenu !!

Les mises à jours

Pour mettre à jours la liste des paquets la commande est la suivante: apt update


Si vous savez pas ce que c'est qu'un paquet, l'ami wiki est là pour vous

Pour mettre à jour les paquets, utilisez la commande suivante: apt upgrade

Si vous désirez rechercher un mot dans vos fichiers, dossier, la commande suivante peut s'avérer très utile: apt search "mot à chercher"

Installer des paquets/Applications téléchargés

La commande apt install "nom du paquet" vous permet de d'installer, via votre terminal, les paquets dont vous avez besoin.


Souvent il vous est demandé votre mot de passe pour confirmer l'installation. Un peu de sécurité ne mange pas de pain!

Pour l'installation de logiciel, la commande dpkg -i nom-du-fichier vous permet d'installer les logiciels ou applciations que vous aurez téléchargés.

Création de dossiers et de fichiers

LA commande pour créer un Dossier est le suivant: mkdir

Pour créer des sous dossiers dans un dossier, la commande mkdir -p va permettre à l'ordinateur de créer un dossier et son sous dossier directement
exemple: mkdir -p Memento/assets. Le dossier "Memento" est créer et son sous dossier "assets" également
